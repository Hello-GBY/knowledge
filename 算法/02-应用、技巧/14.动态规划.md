# 从“爬楼梯”问题说起
> 题目描述：假设你正在爬楼梯。需要 n 阶你才能到达楼顶。
每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 n 是一个正整数。

## 用递归的思想
> f(n) = f(n-1) + f(n-2); f(n-1) = f(n-2) + f(n-3)

```js
let fn = []
function climbStairs (n) {
  if(n == 1) {
    return 1
  }
  
  if(n === 2) {
    return 2
  }

  if(fn[n] ==  undefined) fn[n] = climbStairs(n-1) + climbStairs(n-2)

  return fn[n]
}
```
